# Purpose

Our design system is more than just a set of UI components. It is built to serve a higher purpose by being inclusive, accessible, sustainable, and beneficial to the global community.

## Mission

- **Inclusivity**: Design interfaces that cater to diverse needs, including neurodiverse users and people with disabilities.
- **Sustainability**: Implement environmentally sustainable practices in our development process.
- **Global Impact**: Create solutions that address global challenges and benefit communities around the world.

Learn more about our mission and how we strive to achieve it in our [Purpose Guide](./PurposeGuide.md).
